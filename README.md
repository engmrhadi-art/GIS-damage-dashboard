# Damage Assessment Dashboard Demo

لوحة Web GIS ثابتة باللغة العربية لعرض بيانات تجريبية عن مواقع الأضرار والطرق والأراضي الزراعية. تعمل مباشرة على GitHub Pages ولا تحتاج إلى خادم خلفي أو قاعدة بيانات.

## تنبيه البيانات

جميع ملفات `data/` المرفقة اصطناعية ومخصصة للعرض التقني فقط. لا تمثل تقييمًا رسميًا ولا يجوز استخدامها لاتخاذ القرار.

## التشغيل المحلي

لا تفتح `index.html` مباشرة من نظام الملفات، لأن المتصفح قد يمنع تحميل GeoJSON. شغّل خادمًا محليًا من مجلد المشروع:

```bash
python3 -m http.server 8000
```

ثم افتح `http://localhost:8000`.

## النشر على GitHub Pages

1. أنشئ مستودعًا جديدًا على GitHub.
2. ارفع محتويات هذا المجلد إلى جذر المستودع.
3. افتح `Settings > Pages`.
4. اختر `Deploy from a branch` ثم الفرع `main` والمجلد `/ (root)`.
5. انتظر اكتمال النشر وافتح الرابط الذي يعرضه GitHub.

الملف `.nojekyll` موجود لتقديم الملفات الثابتة كما هي.

## تعديل البيانات

المسارات وأسماء الملفات معرفة في `js/config.js`. الحقول المستخدمة هي:

- مواقع الأضرار: `S`, `M`, `N`, `D` مع هندسة `Point`.
- الطرق: `RoadType`, `Classification`, `Length_m`, `Governorate`, `Municipality`.
- الأراضي الزراعية: `Class`, `Area_km2`, `Governorate`, `Municipality`.

لا تضع بيانات شخصية أو مفاتيح API أو ملفات داخلية في مستودع عام.

## خرائط الأساس

تُعرض خرائط الأساس من OpenStreetMap وEsri وCARTO مع نسب المصدر داخل الخريطة. لا يحتوي المشروع على روابط Google Tiles غير الرسمية.

انظر `THIRD_PARTY_NOTICES.md` لمكونات الطرف الثالث.
